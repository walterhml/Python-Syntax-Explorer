Números, Strings e Operadores

Python oferece suporte a uma variedade de tipos de dados, incluindo números e strings.

Operadores lógicos e condicionais são amplamente utilizados para controlar o fluxo de um programa.


# Exemplos de números
inteiro = 10
flutuante = 3.14
complexo = 2 + 3j

# Exemplos de strings
texto = "Olá, mundo!"
caractere = 'a'

# Operadores lógicos e condicionais
a = 5
b = 3

# Operadores aritméticos
soma = a + b
subtracao = a - b
multiplicacao = a * b
divisao = a / b

# Operadores lógicos
e_logico = (a > 0) and (b < 5)
ou_logico = (a > 0) or (b > 5)
nao_logico = not (a == b)

# Operadores condicionais
if a > b:
    print("a é maior que b")
elif a == b:
    print("a é igual a b")
else:
    print("a é menor que b")

Este exemplo demonstra diferentes tipos de números, como inteiros, flutuantes e complexos, bem como manipulação de strings e exemplos de operadores aritméticos, lógicos e condicionais em Python.

Passo 8: Orientação a Objetos

Python suporta programação orientada a objetos.

Passo 8: Orientação a Objetos (Continuação)

Python suporta herança, encapsulamento e polimorfismo.


# Exemplo de herança
class Animal:
    def __init__(self, nome):
        self.nome = nome

    def som(self):
        pass

class Cachorro(Animal):
    def som(self):
        return "Au au!"

class Gato(Animal):
    def som(self):
        return "Miau!"

# Criando instâncias das subclasses
rex = Cachorro("Rex")
print(rex.som())

garfield = Gato("Garfield")
print(garfield.som())

Passo 9: Métodos Especiais (Mágicos)

Python possui métodos especiais para funcionalidades especiais.


class Numero:
    def __init__(self, valor):
        self.valor = valor

    def __add__(self, outro):
        return self.valor + outro.valor

    def __str__(self):
        return str(self.valor)

# Criando instâncias da classe Numero
numero1 = Numero(5)
numero2 = Numero(10)

print(numero1 + numero2)  # Saída: 15

Passo 10: Decoradores

Decoradores são funções que modificam o comportamento de outras funções.


# Exemplo de decorador
def decorador(funcao):
    def wrapper():
        print("Antes da chamada da função")
        funcao()
        print("Depois da chamada da função")
    return wrapper

@decorador
def funcao_decorada():
    print("Função decorada")

funcao_decorada()

Passo 12: Automação e Teste de Software

Python é amplamente utilizado para automação de tarefas e teste de software.


# Exemplo de automação de tarefas
import os

def listar_arquivos(diretorio):
    for raiz, diretorios, arquivos in os.walk(diretorio):
        for arquivo in arquivos:
            print(os.path.join(raiz, arquivo))

# Listar arquivos em um diretório
listar_arquivos('/caminho/do/diretorio')

Este exemplo utiliza o módulo os para listar todos os arquivos em um diretório.

Além disso, Python também é utilizado para escrever testes automatizados. Aqui está um exemplo usando o módulo unittest:


# Exemplo de teste automatizado
import unittest

def somar(a, b):
    return a + b

class TesteSoma(unittest.TestCase):
    def test_soma_positiva(self):
        self.assertEqual(somar(3, 5), 8)

    def test_soma_negativa(self):
        self.assertEqual(somar(-3, -5), -8)

if __name__ == '__main__':
    unittest.main()

Neste exemplo, criamos testes para a função somar usando o módulo unittest.

Passo 13: Exemplo de Aplicação de Gerenciamento de Estoque

Neste exemplo, vamos criar uma aplicação de gerenciamento de estoque para uma loja fictícia.

Usaremos classes para representar produtos, métodos para adicionar estoque e vender produtos, e uma função principal para testar nossa aplicação.


# Módulo de gerenciamento de estoque

class Produto:
    def __init__(self, nome, preco, quantidade):
        self.nome = nome
        self.preco = preco
        self.quantidade = quantidade

    def __str__(self):
        return f"{self.nome} - R${self.preco:.2f} ({self.quantidade} em estoque)"

    def adicionar_estoque(self, quantidade):
        self.quantidade += quantidade
        print(f"{quantidade} unidades de {self.nome} adicionadas ao estoque.")

    def vender(self, quantidade):
        if self.quantidade >= quantidade:
            self.quantidade -= quantidade
            print(f"{quantidade} unidades de {self.nome} vendidas.")
        else:
            print("Quantidade insuficiente em estoque.")

# Função principal
def main():
    # Criando alguns produtos
    produto1 = Produto("Camiseta", 29.99, 50)
    produto2 = Produto("Calça Jeans", 59.99, 30)
    produto3 = Produto("Tênis", 99.99, 20)

    # Exibindo informações dos produtos
    print("Produtos disponíveis:")
    print(produto1)
    print(produto2)
    print(produto3)

    # Adicionando estoque e vendendo produtos
    produto1.adicionar_estoque(10)
    produto1.vender(5)
    produto2.vender(40)

if __name__ == "__main__":
    main()

Neste exemplo, criamos uma classe `Produto` que representa um item do estoque, com atributos como nome, preço e quantidade em estoque. Também implementamos métodos para adicionar estoque e vender produtos. Na função principal (`main()`), criamos alguns produtos, exibimos suas informações, adicionamos estoque e vendemos produtos.
